# Athlyx

Athlyx is a modern fitness platform that combines workout tracking and nutrition planning in one place. Log workouts, track macros, plan meals, and analyze your progress with powerful analytics. Built to help athletes and fitness enthusiasts train smarter and fuel their bodies effectively.
## Setup

Create a `.env` file and update with your values:

```sh
cp .env.example .env
```

Make sure to install dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```

Locally preview production build:

```bash
# npm
npm run preview
```
